
# Feedback App

This is my first course project from the React Front to Back 2022  
## Run Locally

Clone the project

```bash
  git clone https://github.com/hassancodess/feedback-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
